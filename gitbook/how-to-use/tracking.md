# 🔎 ติดตามสถานะ

ใช้ Tracking Code เพื่อตรวจสอบสถานะของที่คุณแจ้งไว้ได้ตลอดเวลา!

## 🎯 Tracking Code คืออะไร?

**Tracking Code** คือรหัส 6 หลักที่คุณได้รับหลังจากแจ้งของหายหรือของเจอ

**ตัวอย่าง:** `ABC123`, `XYZ789`, `LMN456`

### ใช้ทำอะไรได้บ้าง?

* 🔍 **ตรวจสอบสถานะ** - เช็คว่าของอยู่ในสถานะอะไร
* 📋 **ดูรายละเอียด** - ดูข้อมูลที่แจ้งไว้ทั้งหมด
* 📞 **ดูช่องทางติดต่อ** - เห็นวิธีติดต่อเจ้าของ/ผู้เจอ
* ⏰ **ดูประวัติ** - เห็นวันที่แจ้งและอัปเดต

---

## 📋 ขั้นตอนการติดตามสถานะ

### 🎬 Step 1: เข้าสู่หน้าติดตาม

มี 3 วิธีในการเข้าถึง:

#### วิธีที่ 1: จากเมนูนำทาง
1. คลิกไอคอน **🔎 ติดตาม** จากเมนู Bottom Navigation (มือถือ) หรือ Sidebar (คอมพิวเตอร์)

#### วิธีที่ 2: จากหน้าหลัก
1. คลิกปุ่ม **"ติดตามสถานะ"** ในหน้าแรก

#### วิธีที่ 3: จาก URL โดยตรง
1. เข้า [bd2fondue.vercel.app/tracking](https://bd2fondue.vercel.app/tracking)

### 🔍 Step 2: ใส่ Tracking Code

1. จะเห็นช่องค้นหาตรงกลางหน้าจอ
2. พิมพ์ **Tracking Code** 6 หลัก ที่คุณได้รับ
3. ตัวพิมพ์เล็ก-ใหญ่ไม่สำคัญ (ระบบจะแปลงให้อัตโนมัติ)

**ตัวอย่าง:**
```
ABC123 → ใช้ได้
abc123 → ใช้ได้
Abc123 → ใช้ได้
```

{% hint style="info" %}
**เคล็ดลับ:**
- คัดลอก-วางจากข้อความที่บันทึกไว้
- ถ่ายภาพหน้าจอเก็บไว้เพื่อความสะดวก
- จดไว้ในสมุดหรือโน้ตมือถือ
{% endhint %}

### 🚀 Step 3: คลิกค้นหา

1. คลิกปุ่ม **"ค้นหา"** หรือกด **Enter**
2. รอระบบค้นหา (1-2 วินาที)
3. ระบบจะแสดงผลลัพธ์

## 📊 การอ่านผลลัพธ์

### ✅ กรณีพบรายการ

หน้าจอจะแสดง:

#### 1. 🏷️ Status Badge (แท็กสถานะ)

แสดงสถานะปัจจุบันด้วยสีที่แตกต่างกัน:

| สี | สถานะ | ความหมาย |
|-----|--------|----------|
| 🔵 **สีฟ้า** | **กำลังตามหา** | ยังไม่พบของ ยังคงค้นหาอยู่ |
| 🟢 **สีเขียว** | **เจอแล้ว** | พบของแล้ว รอรับคืน |
| 🟣 **สีม่วง** | **รับคืนแล้ว** | เจ้าของรับของคืนไปแล้ว ✅ |
| ⚪ **สีเทา** | **หมดอายุ** | รายการเก่า หมดอายุแล้ว |

#### 2. 📋 รายละเอียดของ

**สำหรับของหาย (Lost Item):**
* 📦 **ชื่อของ** - ชื่อที่แจ้งไว้
* 🏷️ **ประเภท** - หมวดหมู่ของ (กระเป๋าสตางค์, โทรศัพท์, ฯลฯ)
* 📝 **คำอธิบาย** - รายละเอียดเพิ่มเติม
* 📍 **สถานที่ทำหาย** - ตำแหน่งที่ทำหาย
* 📅 **วันที่ทำหาย** - วันที่เสียของ

**สำหรับของเจอ (Found Item):**
* 📸 **รูปภาพ** - รูปของที่เจอ
* 📝 **คำอธิบาย** - ลักษณะของ
* 📍 **สถานที่เจอ** - ตำแหน่งที่พบของ
* 📅 **วันที่เจอ** - วันที่พบ
* 🏢 **สถานที่นำส่ง** - ที่เก็บของไว้ (ห้องธุรการ, ห้องสมุด, ฯลฯ)

#### 3. 📞 ช่องทางติดต่อ

แสดงช่องทางที่เจ้าของ/ผู้เจอระบุไว้:

* 📱 **เบอร์โทรศัพท์** - คลิกเพื่อโทร (ในมือถือ)
* 💬 **LINE ID** - คัดลอกเพื่อเพิ่มเพื่อน
* 📷 **Instagram** - คัดลอกชื่อผู้ใช้
* 📘 **Facebook** - คัดลอกชื่อ
* 📧 **Email** - คลิกเพื่อส่งอีเมล

{% hint style="success" %}
**วิธีติดต่อ:**
1. เลือกช่องทางที่สะดวก
2. คัดลอกข้อมูลติดต่อ
3. เปิดแอปนั้นๆ และติดต่อ
4. บอก Tracking Code เพื่อยืนยัน
{% endhint %}

#### 4. ⏰ ข้อมูลเวลา

* 📅 **วันที่แจ้ง** - วันที่สร้างรายการ
* 🔄 **อัปเดตล่าสุด** - วันที่มีการเปลี่ยนแปลงครั้งล่าสุด

---

### ❌ กรณีไม่พบรายการ

ถ้าระบบหารายการไม่เจอ จะแสดงข้อความ:

> **"ไม่พบรายการ"**
>
> ไม่พบรายการที่มี Tracking Code นี้ กรุณาตรวจสอบอีกครั้ง

**สาเหตุที่เป็นไปได้:**

1. ❌ **พิมพ์ Code ผิด** - ตรวจสอบอีกครั้งว่าพิมพ์ถูกต้องหรือไม่
2. ❌ **Code หมดอายุ** - รายการถูกลบหรือหมดอายุแล้ว
3. ❌ **Code ไม่มีอยู่จริง** - อาจจำผิดหรือรับผิด

**วิธีแก้:**

* ✅ ตรวจสอบ Code อีกครั้ง
* ✅ ดูจากข้อความ/ภาพหน้าจอที่เซฟไว้
* ✅ ลองค้นหาในหน้า "รายการ" (ถ้าล็อกอินด้วยบัญชีเดิม)
* ✅ ติดต่อผู้ดูแลระบบถ้าแน่ใจว่า Code ถูกต้อง

## 🎯 สถานการณ์การใช้งาน

### 📱 สถานการณ์ที่ 1: คุณทำของหาย

1. แจ้งของหายในระบบ → ได้ Tracking Code: `ABC123`
2. เซฟ Code ไว้
3. ทุกวันใช้ Code เช็คสถานะ
4. วันหนึ่ง สถานะเปลี่ยนเป็น **"เจอแล้ว"** 🎉
5. เห็นช่องทางติดต่อผู้เจอ
6. ติดต่อไปรับของ
7. รับของคืนแล้ว แจ้งผู้ดูแลอัปเดตเป็น **"รับคืนแล้ว"**

---

### 📸 สถานการณ์ที่ 2: คุณเจอของ

1. แจ้งเจอของในระบบ → ได้ Tracking Code: `XYZ789`
2. เซฟ Code ไว้
3. นำของไปส่งที่ห้องธุรการ
4. เจ้าของเห็นรายการและติดต่อมาทาง LINE
5. บอกว่านำส่งไว้ที่ไหน
6. เจ้าของไปรับของ
7. เช็ค Code อีกครั้ง สถานะเป็น **"รับคืนแล้ว"** ✅

---

### 🔍 สถานการณ์ที่ 3: คุณเห็นประกาศของเจอ

1. เพื่อนบอกว่ามีคนเจอกระเป๋าสตางค์
2. เพื่อนให้ Tracking Code: `LMN456`
3. เข้าหน้าติดตาม ใส่ Code
4. เห็นรูปและคำอธิบาย
5. เช็คว่าเป็นของตัวเองหรือไม่
6. ถ้าใช่ → ติดต่อผู้เจอตามช่องทางที่ระบุ
7. ไปรับของตามสถานที่นำส่ง

---

## 💡 เคล็ดลับการใช้งาน

### ✅ DO's (ควรทำ)

* 💾 **เซฟ Code ทันที** - พอได้รับ Code ให้เซฟทันที
* 🔄 **เช็คสม่ำเสมอ** - เช็คสถานะเป็นประจำ
* 📱 **แชร์อย่างปลอดภัย** - แชร์เฉพาะกับคนที่เกี่ยวข้อง
* 📞 **ติดต่อสุภาพ** - ติดต่อด้วยความสุภาพและขอบคุณ
* ✅ **อัปเดตสถานะ** - แจ้งเมื่อได้รับของคืนแล้ว

### ❌ DON'T (ไม่ควรทำ)

* 🚫 **อย่าแชร์ Code ทั่วไป** - อาจมีคนอ้างเป็นเจ้าของ
* 🚫 **อย่าลืม Code** - อาจไม่สามารถติดตามได้
* 🚫 **อย่าเช็คบ่อยเกินไป** - สถานะไม่เปลี่ยนแปลงทันที
* 🚫 **อย่ารบกวนผู้เจอ** - ติดต่อแค่พอดีพอเหมาะ

## ❓ คำถามที่พบบ่อย

### Q: Tracking Code หายแล้วทำยังไง?

**A:** 
1. เข้าหน้า **"รายการ"** ในเมนู (ต้องล็อกอินด้วยบัญชีเดิม)
2. ดูรายการที่คุณแจ้งไว้ทั้งหมด
3. คลิกรายการที่ต้องการ จะเห็น Tracking Code
4. ถ้ายังหาไม่เจอ → ติดต่อผู้ดูแลระบบ

### Q: ทำไมสถานะไม่เปลี่ยน?

**A:** สถานะจะเปลี่ยนเมื่อ:
- มีคนแจ้งเจอของที่ตรงกับของคุณ
- Admin จับคู่ของหายกับของเจอ
- Admin อัปเดตสถานะด้วยตนเอง

อาจต้องใช้เวลา รอติดตามต่อไปค่ะ

### Q: สามารถดูรายการอื่นได้ไหม?

**A:** ได้ค่ะ ไปที่หน้า **"รายการ"** เพื่อดูรายการทั้งหมดในระบบ

### Q: Code ใช้ได้กี่ครั้ง?

**A:** ไม่จำกัดครั้ง ใช้ติดตามได้ตลอดเวลาจนกว่ารายการจะถูกลบ

### Q: มีคนติดต่อมาแต่ไม่แน่ใจว่าเป็นเจ้าของจริง?

**A:** 
1. ถามคำถามเพื่อยืนยันความเป็นเจ้าของ
2. เช่น ถามรายละเอียดเพิ่มเติมที่คุณไม่ได้ระบุในระบบ
3. ถามให้บอก Tracking Code
4. ถามว่ามีอะไรพิเศษ/ลักษณะเฉพาะ
5. ถ้าไม่แน่ใจ → บอกให้ไปติดต่อผู้ดูแลระบบแทน

### Q: ถ้ารายการถูกลบแล้ว Code ยังใช้ได้ไหม?

**A:** ไม่ได้ค่ะ ถ้ารายการถูกลบ Code จะไม่สามารถค้นหาเจออีกต่อไป
